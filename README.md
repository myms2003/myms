# MYMS

MYMS is a premium SaaS-style commercial digital platform for real businesses.

Core positioning:

> MYMS turns local businesses and independent professionals into digital systems that generate trust, inquiries, bookings and sales.

## What is in this repository

- `apps/landing`: main marketing/product landing built with React, TypeScript, Vite, Tailwind, GSAP and React Three Fiber
- `docs/strategy`: positioning and commercial framing
- `docs/branding`: design-system and brand rules
- `docs/content`: launch and social content
- `docs/research`: customer pains, objections and signal patterns
- `agents`: future agent workflows and automation context
- `packages`: reserved shared layers for UI, utils and types

## Commands

From `apps/landing`:

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Product model

MYMS is not a web design agency.

MYMS builds commercial digital systems such as:

- landing pages
- booking systems
- CRM
- dashboards
- digital business cards
- WhatsApp contact flows
- forms
- local SEO and maps
- analytics
- staff profiles
- lead tracking

## Product packs

- `Presencia Pro`: trust-first system for professionals and small businesses
- `Turnos Pro`: booking-first system for appointment-based businesses
- `Ventas Pro`: CRM and pipeline-first system for commercial teams
- `Local Pro`: local-conversion system for physical businesses

## Current goal

The first production-ready foundation of the MYMS landing: premium visual direction, strong positioning, reusable sections, 3D hero system, GSAP reveals, SEO structure and WhatsApp conversion flow.
