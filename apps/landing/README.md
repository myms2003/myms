# apps/landing

Production foundation for the MYMS marketing and product landing.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- GSAP
- Framer Motion
- Three.js
- React Three Fiber
- Drei

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Notes

- The hero and showcase scenes use MYMS design tokens and reduced-motion safeguards.
- WhatsApp CTAs use the placeholder MYMS number from `src/lib/site.ts` and should be replaced when production data is available.
